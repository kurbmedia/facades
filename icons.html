Facades

Front-end awesomeness, with Compass and SASS

Accessible Font-Face Icons class names optional

Facades includes a @font-face friendly icon set, using the unicode private range (U+e00+) for all of its glyphs. This way you can join in on all of the font-icon madness like the other cool kids do, but in a screen-reader friendly manner. Since icons aren't mapped to standard letters/numbers/punctuation, a icon mixin is included.

If you'd like to have all of the classes below, you can include the icon-classes mixin in your top-level stylesheet.

// The icon-classes takes a single argument which represents the selector to 'scope' all icons to.
@include icon-classes("span.icon")
